## Bank Loan Analysis – Power BI Dashboard

### Project Overview
This project analyzes bank loan data to understand lending patterns, loan performance, and borrower behavior. 
The analysis was done using SQL for data preparation and Power BI for visualization and insights.

### Tools Used
- SQL Server – Data analysis & KPI calculations
- Power BI – Dashboard creation & DAX
- DAX – Time intelligence (MTD, MoM, YTD)

### Key Insights
- Good loans account for **86.2%** of total applications
- December shows the highest loan activity
- 36-month loans dominate with **73.2%** of applications
- Debt consolidation is the top loan purpose
